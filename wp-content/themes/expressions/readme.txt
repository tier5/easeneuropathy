Expressions is a full featured responsive theme that looks great on full size screens, 
tablets and mobile devices. Users have incredible flexibility to create the website 
that suits your style, with extensive theme options, post metabox options, and page 
metabox options. Comprehensive user documentation is available at the demo site.

== Description ==

	Expressions offers  is a full featured Wordpress theme offering the user many 
	options to control the look of the theme, and many options for special pages 
	and content.

	1) Header Options
		- logo display is optional, left or center of header
		- blog title optionally displayed in the center of the header
		- blog description is optionally displayed in the center of the header
		- widgetized area on right side of header for social icons and text
		- widget on the left side of the header	is activated when logo, title 
		  and description are not used to give the user full customization potential
		- Full header banner images permitted
		
	2) Menu Options
		- left,right,center
		- full width borders, no borders, menu only borders in the center menu option.
		
	3) Static Home Page Options
		- Feature area with 9 nivo slider options, 3 mosaic options, 3 single image options
		- 8 layout sections below the feature section that are all optional, giving the user 
		  great layout flexibility for the most important page on your site
		- Feature images can link to feature post, no link, or custom link
		- New: Section 8 contains 4 service boxes
		
	4) Blog Options
		- post formats: aside, audio, image, quote, status, video, standard
		- full width blog option, sidebar blog option, or include with feature section
		- option to display single posts without a sidebar
		- options to exclude post meta:date,author,category,tags
		- sticky posts displayed at the top of your blog
		- post format icons optionally included
		
	5) Footer
		- Footer display optional
		- Footer can be 3 or 4 columns
		- copyright strip optional
		- left, center, and right sections in copywrigt strip
		
	6) Sidebars
		- 404, about, blog, contact, default, sidebar_1 to sidebar_6
		- all sidebars can be positioned left or right.
		- New: Contact Widget, use this widget instead of a Contact Page
		- New: Author Widget, Author Favicon, Bio, Website link
		
	7) Social Widget 
		- Delicious, Digg, Facebook, Google+, Linkedin, Myspace, Pinterest, Rss, Tumblr, Twitter
		- 24x24px or 32x32px
		- select only the social links you want for each widget instance
		- New : added Flickr, Instagram, Vimeo, Yelp, Youtube
		- New : Added option to use font icons
		
	8) Optional comment and contact page captcha, color or black and white
	
	9) Preset skins : black, blue, gray, block gray, block brown
	
	10) Child Theme Friendly
	
	11) Selection of standard web fonts google fonts, and @fontface fonts available for 
	    headers, and body sections separately.

	12) Templates
	   - Static Home Page
	   - Featured Home Page c/w blog
	   - Full Width Blog
	   - Full Width Page
	   - Full Width with feature slider options
	   - Archive
	   - Siteplan
	   - Contact template with custom captcha and widgetized sidebar
	   - Unlimited Portfolio Templates, that you can customize with 
	     up to 4 columns, c/w post content or feature media text options

	13) PageNavi ready
	
	14) Font Icons
	    - NEW: user can select font icons for the entire theme instead of images.

== Installation ==

1) Upload to the themes directory of your wordpress setup. Ensure all sub directories are maintained.

2) Activate the theme through the Wordpress Admin panel under "Appearance" and "themes"
