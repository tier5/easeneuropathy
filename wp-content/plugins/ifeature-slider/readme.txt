Creates a 2 slide image slider and the shortcode required to place it anywhere in your site.

== Description ==

CyberChimps's free <a href="http://cyberchimps.com/store/ifeature-slider/">iFeature Slider</a> is a popular free WordPress slider that's truly responsive. Create a new slider through the iFeature Slider menu, where you can upload your images or use ones you have in your media library.

Once you have added your images you have a few more options for your new iFeature Slider.

* Change the slider delay in seconds
* Change the transition time in seconds
* Select to auto slide

Publish your slider and copy and paste the shortcode anywhere in your site to see it in action

If you have any questions or suggestions, CyberChimps would love to hear from you - Get in touch via <a href="http://cyberchimps.com/forum/free/plugins/">CyberChimps Forums</a>


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ifeatureslider-free` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the supplied shortcode on any page or post in your site
